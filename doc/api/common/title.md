说明：`title` 属性，替代旧属性`text`

格式：

```d
title:{
    text: " ",
    subtext: " ",
    top: 'center',
    left: 'center',
    itemGap: 8,
    textStyle: {
        color: '#191919',
        fontSize: 28,
        lineHeight: 28,
        fontWeight: 'normal',
    },
    subtextStyle: {
        color: '#bbbbbb',
        fontSize: 16,
        lineHeight: 24,
        fontWeight: 'normal',
    },
}
```
