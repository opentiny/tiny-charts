格式：

```d
 marginWidth: [
    {
      left: 50,
      right: 50,
    },
    {
      left: 100,
      right: 100,
    },
    {
      left: 50,
      right: 50,
    },
  ],
```

默认值：`marginWidth: [
    {
      left: 50,
      right: 50,
    },
    {
      left: 100,
      right: 100,
    },
    {
      left: 50,
      right: 50,
    }
  ]`

说明：有几个阶级配置几个对象，left 控制左边，right 控制右边
