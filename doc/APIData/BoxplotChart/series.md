格式：

```d
series:[
        {
            name: 'category0',
            type: 'boxplot',
            datasetIndex: 3
        },
        {
            name: 'category1',
            type: 'boxplot',
            datasetIndex: 4
        },
        {
            name: 'category2',
            type: 'boxplot',
            datasetIndex: 5
        }
    ]
```

详细参数解释见： https://echarts.apache.org/zh/option.html#series-sunburst.type
